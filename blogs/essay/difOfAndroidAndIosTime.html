# IOS和Android的生成时间形式不同

# IOS

let date = new Date('2021/11/12 00:00:00')      //ios的时间是这种斜杠的

如果在ios端使用类似这种

let date = new Date('2021-11-12 00:00:00')      //则会报错,Invalid Date

# Android

let date = new Date('2021-11-12 00:00:00')      //android的时间是这种横杠的
