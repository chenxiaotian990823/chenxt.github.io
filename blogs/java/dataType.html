# 基本数据类型

基本数据类型描述的是一些具体的数字单元，例如:1、1.1

# 数值型

整型

  • byte --默认值: 0 --1个字节
  • short --默认值: 0 --2个字节
  • int --默认值: 0 --4个字节
  • long --默认值: 0 --8个字节
  • 0b开头是二进制
  • ox是十六进制
  • 0是八进制

浮点型

  • float --默认值: 0.0 --4个字节
  • double --默认值: 0.0 --8个字节
  • 涉及到钱，用BigDecimal

# 布尔型

TIP

  • boolean --默认值: false --占1位

# 字符型

TIP

  • char --默认值: '\u0000'
  • ascii编码
  • utf-8
  • Unicode

使用参考原则

  • 如果是描述数字，首选的一定是int(整数),double(小数)
  • 如果要进行数据传输或者进行文字编码转换，使用byte类型(二进制处理操作)
  • 处理中文的时候，最方便的操作是字符char来操作
  • 描述内存或文件大小、描述表的主键列，可以使用long
